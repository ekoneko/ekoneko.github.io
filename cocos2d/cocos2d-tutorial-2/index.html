cocos2d-html 学习笔记 3: 第一个游戏--潜艇大战

用cocos2d-html5做的第一个小作品:潜艇大战.

这复刻于我小学时很喜欢的一个小游戏, 主要开发目的是熟悉cocos2d的工程结构与实践.

制作时间控制在刚好一个礼拜,每天coding .5~2小时.周末有些感冒没有动工(本来想实现计分板和爆炸的动画效果的,因为身体原因放弃了). 因此开发时间大约是5小时左右,素材都来源于deviantart.com. 因为当初做的时候就定下一周搞定的规矩,就游戏而言完全没有可玩性吧233.

另外这个项目的完成时间大约是3月初, 我拖了一个月才来记录这篇笔记主要是因为当时生病了一个星期,其后稍微有些倦怠了,忏悔下…

大致的游戏规则是这样:

游戏开始时按F2进入新游戏.

玩家控制一艘船可以在水面上按左右键移动,空格键丢下炸弹

水面下会有潜艇(就算我用的模型是⑨也依然是潜艇)来回移动, 如果玩家的船所投下的炸弹与潜艇触碰潜艇就会消失.

游戏规则

原版游戏的话会有多种船,并且敌人有各自的攻击方式,当得分到一定时可以进入下一关,不过这里简化省略了(其实是懒)

cc.Application.extend({
    config: document['ccConfig'],
    ctor: function() {
        this._super();
        // ...
    },
    applicationDidFinishLaunching: function() {
        var self = this;
        if(cc.RenderDoesnotSupport()){
            // Browser doesn't support WebGL
        }
        // initialize director
        var director = cc.Director.getInstance();
        // turn on display FPS
        director.setDisplayStats(this.config['showFPS']);
        // set FPS. the default value is 1.0/60 if you don't call this
        director.setAnimationInterval(1.0 / this.config['frameRate']);
        //load resources
        cc.LoaderScene.preload([], function () {
            director.replaceScene(new this.startScene());
        }, this);
        return true;
    }

size(__width, __height), cc.TEXT_ALIGNMENT_CENTER);
welcomeWord.setPosition(__width / 2,0);
LayerWelcome = cc.Layer.extend({
    ctor: function() {
        this._super();
        this.setKeyboardEnabled(true);
        this.addChild(welcomeWord);
    },
    onKeyDown: function(keyCode) {
        if (keyCode === 113) {
            // 当按下F2时游戏开始

    }

LayerPlayer.js

玩家控制的船

var ship = cc.Sprite.create('res/player.png');
ship.setPosition(__width / 2, __height - 105);
cc.Layer.extend({
    __downButtons: [],
    ctor: function() {
        this._super();
        this.setKeyboardEnabled(true);
        this.scheduleUpdate();
        this.appendChild(ship);
    },
    update: function() {
        this._super();
        // 为了避免 keydown 在多键按下时的响应问题
        for (var i = this.__downButtons.length - 1; i >= 0; i--) {
            this.onKeyPress(this.__downButtons[i]);
        };
    },
    onKeyDown: function() {
        if (this.__downButtons.indexOf(keyCode) === -1) {
            this.__downButtons.push(keyCode);
        }
        if (keyCode === 32) { // SPACE
            game.dropBoom(ship.getPositionX());
    },
    onKeyUp: function(keyCode) {
        var index = this.__downButtons.indexOf(keyCode);
        if (index !== -1) {
            this.__downButtons.splice(index);
        }
    },
    onKeyPress: function(keyCode) {
        var x;
        if (keyCode === 37) {
            // LEFT
            x = ship.getPositionX() - 5;
        } else if (keyCode === 39) {
            // RIGHT
            x = ship.getPositionX() + 5;
        } else {
            return;
        }
        x = Math.max(70, x);
        x = Math.min(__width - 70, x);
        ship.setPositionX(x);
    }
})

这里就是遇到如果左右键同时按下会导致一段时间没响应的问题,于是自己实现了keyPress的事件

LayerEnemy.js

LayerEnemy = cc.Layer.extend({
    ctor: function() {
        this._super();
        this.scheduleUpdate();
    },
    createEnemy: function(x, y, direct) {
        var sprite = cc.Sprite.create('res/9.png');
        sprite.setPosition(x, y);
        if (direct === 1) {
            sprite.setRotation(180);
        }
        return sprite;
    }
});
